## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/CODSOFT_TASK1.git
```

Go inside project

```bash
cd CODSOFT_TASK1
```

Install dependencies

```bash
npm install
```

Run the project

```bash
npm run dev
```
